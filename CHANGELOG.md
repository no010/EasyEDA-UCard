# Changelog

## 0.1.0

### 新增

- 💳 **导入名片模板**：将内置 epro2 模板导入当前工程；内置模板优先从扩展本地文件加载，加载失败时自动从 GitHub 获取；支持用户替换为本地 epro2 文件
- ✏️ **替换名片文本**：读取当前 PCB 所有文本图元，显示在列表中供逐一编辑，批量写回
- 📱 **生成名片二维码**：
  - vCard 模式：填写联系信息生成符合 RFC 2426 标准的 vCard 3.0 二维码（微信/手机扫码可保存联系人）
  - 图片模式：上传或粘贴图片转为铜皮填充二维码
- 🌐 双语界面（简体中文 / English）
- ⚡ 仅在 PCB 编辑器可用
