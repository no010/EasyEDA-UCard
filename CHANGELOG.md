# 0.1.0

## 新增

- 🎉 初始版本发布
- 💳 支持模板化 PCB 名片生成（`AURORA` / `MONO`）
- 👨‍💻 开发者专属字段：GitHub、技术栈、个性化 Slogan
- 📱 智能 QR 码生成（优先使用成熟库，自动降级到本地实现）
  - 主策略：`qrcode` npm 库（成熟、符合 ISO 标准）
  - 降级策略：本地 QR 编码实现（ISO/IEC 18004 子集）
  - 输出格式：vCard 3.0（扫码即可保存联系人）
- ✨ 智能文字渲染：优先使用文字图元，自动降级到点阵焊盘
- 🛡️ 板框智能检测：自动识别现有板框并提示用户选择
- 🌐 完整支持中文和英文界面
- ⚡ 在 PCB 编辑器中可用

## 技术特性

- 基于 EasyEDA Pro API v3.0.0+
- TypeScript 开发，类型安全
- 模板化架构设计
- 渐进式降级策略（QR 码、文字渲染）
- 无外部网络依赖（纯本地运行）
